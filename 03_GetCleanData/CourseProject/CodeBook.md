## Code Book - HAR Tidy Data Set

##### This code book refers to tidydata.txt file that was generated from Human Activity Recognition Using Smartphones Dataset, downloaded at the site (http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones)

> subject 
- identifies the subject who performed the activity for each window sample. Its range is from 1 to 30.

> activity
- activity names performed by each person, as follows:
1. WALKING
2. WALKING_UPSTAIRS
3. WALKING_DOWNSTAIRS
4. SITTING
5. STANDING
6. LAYING
> timeBodyAcc-mean()-X
